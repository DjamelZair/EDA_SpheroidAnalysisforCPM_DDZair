About this analysis

This EDA pipeline systematically characterises a unique biomedical imaging dataset of Chronic Lymphocytic Leukaemia (CLL) spheroids imaged at multiple timepoints under various drug treatment conditions.

The dataset includes 51 manually annotated brightfield image–mask pairs (ground truth), a time-series pool of VID experiment images with instance segmentation masks, and a raw IncuCyte archive of approximately 102,000 files spanning BTK inhibitor, drug screen, and refractory patient experiments.

A key challenge motivating this analysis is the extreme label scarcity: the annotated pool represents only 0.05% of the full imaging archive, motivating semi-supervised and self-supervised learning strategies for downstream spheroid segmentation and drug-response modelling.

📋
Data Provenance
Images exported from Sartorius IncuCyte S3 live-cell imager. Two data transfers covering BTK, drug-screen, NK, T-cell, and refractory experiment classes.
🧬
Patient Cohort
Peripheral blood CLL patient samples from three clinical experiment series. Demographics, cytogenetics (IGHV, del13q14, del17p13…), and clinical lab values linked via Excel metadata.
📈
Evaluation Metrics
Binary segmentation: Dice, precision, recall. Instance counting: IoU-matched object recall. Metadata: fill-rate completeness per domain group.
🛠
Pipeline Stack
Python · NumPy · pandas · matplotlib · scikit-image · SAM2 (Meta). All outputs reproducible from a single run_all.py call.
